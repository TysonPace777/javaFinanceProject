# Overview

I am a Software Developer pursuing a Bachelor's from BYUI. I love to code and create programs that do all sorts of tasks for you.

The Java Finance project has a menu of options for the user to deposit, withdraw, view their balance, write history to a file, and read previous saved history from a file.

I created this software so anybody can have an easy way to view and manage their finances without needing to write anything down or do any math.

https://youtu.be/m795gbYqc6Q

# Development Environment

This program was produced using vscode and the Microsoft Java extension pack as well as GitHub for hosting.

The program utilizes Java which is an Object Oriented Programming Language, and the Java collection framework.
